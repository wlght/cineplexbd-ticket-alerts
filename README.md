# 🎬 CineplexBD Ticket Alerts

**CineplexBD Ticket Alerts** is a personal project built to ensure I never miss the moment movie tickets go on sale on CineplexBD.

The application automatically monitors ticket availability at regular intervals and sends instant notifications to a Discord server whenever sales open for a selected movie. Powered by Python and GitHub Actions, it runs entirely in the cloud, no dedicated server required.

Although this project was created for my own use, anyone is welcome to use or build upon it. Simply fork the repository, configure your own Discord webhook, customize the movies you'd like to track, and enable GitHub Actions. The repository includes everything you need to get started, with setup instructions available in the README.

If you'd rather receive alerts without setting everything up yourself, you're welcome to join the community Discord server where notifications are shared:
**https://discord.gg/BnpMJrKuv7**
